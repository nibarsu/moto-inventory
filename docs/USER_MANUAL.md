# Moto Inventory 使用手冊

## 1. 系統定位

本系統提供機車行進銷存後台管理，現階段已支援：

- 品牌管理
- 商品分類管理
- 倉庫管理
- 供應商管理
- 客戶管理
- 零件商品管理
- 整車商品管理
- 庫存查詢
- 庫存調整
- 進貨單
- 進貨單明細
- 進貨入庫
- 平均成本查詢
- 銷貨單
- 銷貨單明細

## 2. 登入

1. 開啟登入頁
2. 輸入 Email 與密碼
3. 驗證成功後進入 Dashboard

## 4. 基本資料模組

以下模組皆支援列表、新增、編輯、刪除：

- 品牌
- 商品分類
- 倉庫
- 供應商
- 客戶
- 零件商品
- 整車商品

## 5. 庫存模組

### 5.1 庫存查詢

- 可查看零件與整車在各倉庫的目前庫存
- 顯示類型、商品代碼、名稱、倉庫、庫存量

### 5.2 庫存調整

1. 進入 `庫存調整`
2. 選擇商品類型
3. 選擇商品
4. 選擇倉庫
5. 輸入調整後庫存
6. 填寫備註後送出

系統會自動記錄：

- 調整前數量
- 調整後數量
- 差異數量
- 異動人員

## 6. 進貨模組

### 6.1 建立進貨單

1. 進入 `進貨單管理`
2. 點選 `新增進貨單`
3. 填寫供應商、倉庫、日期、狀態
4. 儲存表頭

### 6.2 維護進貨單明細

1. 進入目標進貨單
2. 點選 `明細管理`
3. 新增或編輯明細
4. 選擇商品類型與商品
5. 輸入數量與單價

系統會自動更新進貨總額。

### 6.3 進貨入庫

1. 進入 `進貨入庫`
2. 選擇對應進貨單
3. 輸入本次入庫數量與成本
4. 儲存後系統自動：
   - 增加庫存
   - 建立庫存異動
   - 更新已收貨數量
   - 更新最近成本與平均成本

## 7. 銷貨模組

### 7.1 建立銷貨單

1. 進入 `銷貨單管理`
2. 點選 `新增銷貨單`
3. 填寫客戶、倉庫、日期、狀態
4. 儲存表頭

### 7.2 維護銷貨單明細

1. 進入目標銷貨單
2. 點選 `明細管理`
3. 點選 `新增明細`
4. 選擇商品類型：
   - `part`
   - `vehicle`
5. 選擇對應商品
6. 輸入數量與單價
7. 需要時填寫備註
8. 儲存

系統會自動計算：

- 小計
- 銷貨總額

## 8. 平均成本查詢

- 可依類型查看零件與整車的平均成本
- 顯示最近成本、平均成本、售價與庫存資訊
